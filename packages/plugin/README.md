# @aura/plugin

Claude Code plugin for auto-generating video demos of web app changes.

## Installation

```bash
claude --plugin-dir ./packages/plugin
```

Or add marketplace (once published):
```
/plugin marketplace add tawsifkamal/aura
```

## Usage

After making changes to a web app:
```
/record-demo
```

## Requirements

- Playwright MCP configured in Claude Code
